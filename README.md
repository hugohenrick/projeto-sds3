# Projeto Dashboard de Vendas

Projeto promovido pela DevSuperior.

# Tecnologias utilizadas
## Back End
- Java
- Spring Boot
- JPA / Hibernate
- Maven

## Front End
- HTML / CSS / JS / TypeScript
- ReactJS
- React Native
- Apex Charts

## Implantação em produção
- Back end: Heroku
- FrontEnd Web: Netlify
- Banco de dados: PostgreSQL
